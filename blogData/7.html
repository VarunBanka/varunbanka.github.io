How to Create a Windows 10 Bootable USB Drive from Ubuntu?

Are you looking to create a bootable USB drive for installing Windows 10 on your computer, but don’t have access to a Windows machine? Fortunately, you can easily create a Windows 10 bootable USB drive from Ubuntu using the WoeUSB tool. In this guide, we’ll show you how to do just that in a few simple steps.

Step 1: Download the Windows 10 ISO

The first step is to download the Windows 10 ISO image from Microsoft’s website. You can download it from this link.

Step 2: Install the WoeUSB tool

WoeUSB is a free and open-source tool that allows you to create Windows bootable USB drives from Ubuntu. To install WoeUSB, open the Terminal app by pressing Ctrl + Alt + T on your keyboard. Then run the following command:

sudo add-apt-repository ppa:nilarimogard/webupd8

This command adds the WebUpd8 PPA repository to your system, which contains the latest version of WoeUSB.

Next, update your system’s package list by running:

sudo apt-get update

Finally, install WoeUSB by running:

sudo apt-get install woeusb

Step 3: Format the USB drive

Before creating the Windows 10 bootable USB drive, you need to format the USB drive in the NTFS file system.

Note: This step will delete all data on your USB drive. Make sure you have backed up all important data before proceeding.

To format the USB drive, insert the USB drive into your computer and open the Disks app by searching for “Disks” in the Applications menu.

In the Disks app, select the USB drive from the list of disks and partitions. Then click on the gear icon and select “Format Partition”.

Choose “NTFS” as the file system and click “Format”. Wait for the formatting process to complete.

Step 4: Create the Windows 10 bootable USB drive

Once the USB drive is formatted, you can create the Windows 10 bootable USB drive using the WoeUSB tool.

Open the WoeUSB tool by searching for “WoeUSB” in the Applications menu.

In the WoeUSB tool, select the Windows 10 ISO file that you downloaded in Step 1.

Select the USB drive that you formatted in Step 3 as the target drive.

Click on the “Install” button to start creating the bootable USB drive.

The process may take some time depending on the speed of your USB drive and your computer. Once the process is complete, you should have a bootable USB drive with the Windows 10 installation files.

That’s it! You have now created a Windows 10 bootable USB drive from Ubuntu using the WoeUSB tool. You can now use this USB drive to install Windows 10 on a computer.

In conclusion, creating a bootable Windows 10 USB drive from Ubuntu is a straightforward process that anyone can do with the right tools and instructions. With this guide, you can easily create a Windows 10 bootable USB drive from Ubuntu and install Windows 10 on your computer without the need for a Windows machine.
