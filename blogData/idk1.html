A Guide to Installing Arch (EndeavourOS) in Dual Boot Configuration

Introduction

EndeavourOS is a user-friendly Arch-based Linux distribution designed to make the Archmore accessible to a wider audience. This guide will walk you through the steps to install EndeavourOS in a dual-boot configuration alongside your existing operating system.

Let’s Dive In, Step 1: Download EndeavourOS ISO

  • Visit the EndeavourOS website and download the ISO file by clicking here.
  • Ensure that you have a stable internet connection as the ISO file is approximately 2GB in size.

Step 2 — Make Bootable USB flash drive

  • Download Rufus , a free and open-source tool for creating bootable USB drives.
  • Insert your USB drive into your computer. Be sure to back up any data on the drive as Rufus will format it during the process.
  • Launch rufus and click on the “Select” button next to “Boot selection” and choose the EndeavourOS ISO file you downloaded.

  • Click on the “Start” button in Rufus to begin creating the bootable USB drive.
  • Rufus will format your USB drive and copy the ISO contents to it. Wait for the process to complete.

Step 3 — Disable BitLocker

“BitLocker is the Windows encryption technology that protects your data from unauthorized access by encrypting your drive and requiring one or more factors of authentication before it will unlock it. Windows will require a BitLocker recovery key when it detects a possible unauthorized attempt to access the data.”

To disable Bitlocker on Windows 10, navigate to Start > Settings > Update & Security > Device encryption, then choose Turn Off.

For Windows 11, the method varies depending on the version. As of now, it appears that the option has been removed from both the Control Panel and the Settings app. Therefore, it’s recommended to bypass this step for now and enter the code when prompted. To locate the code, refer to the provided guide by clicking here.

Step 4 — Disable Secure Boot

“Secure Boot is an important security feature designed to prevent malicious software from loading when your PC starts up (boots).” However in order to install Linux, we will need to disable it.

To do so, Power off your PC, then power it back on. Immediately start pressing the BIOS key, which varies depending on your motherboard or laptop brand (try doing a google search to find out yours). 

Image source: Click here

Once in the BIOS settings, navigate to the Security tab using the arrow keys. Locate the Secure Boot option and press Enter to select it. Choose to disable Secure Boot. This step is necessary because Secure Boot restricts the boot process to signed operating systems only. After disabling Secure Boot, navigate to the Exit tab, select ‘Save and Exit’ to apply the changes.
